# 呀呐 YANA (You Are Not Alone)

## 项目简介

**呀呐 YANA** (原名 Still Alive) 是一款专注于独居人士安全监测的 Android 应用。它通过监测用户的日常签到行为（手动或自动），在用户长时间未签到（可能发生意外）时，自动向预设的紧急联系人发送包含位置信息的求助短信。

"YANA: You Are Not Alone" —— 你不孤单。

## 主要功能

*   **每日签到**：
    *   **手动签到**：在首页点击大大的签到按钮即可。
    *   **自动签到**：支持开启“自动模式”，当监测到屏幕解锁时自动完成当日签到，无需手动操作。
*   **紧急报警机制**：
    *   用户可设置“断签报警阈值”（如连续 3 天未签到）。
    *   应用在后台运行监测服务，一旦超过阈值，自动触发报警流程。
    *   **自动发送短信**：向所有紧急联系人发送求助短信。
    *   **信息包含**：短信内容包含用户姓名、连续未签到天数、预设联系地址以及**实时定位信息**（支持 GPS/网络定位，带超时自动降级策略）。
*   **数据管理**：
    *   **本地存储**：所有数据（签到记录、联系人、设置）均存储在本地（Room 数据库 + DataStore），保护隐私。
    *   **导入导出**：支持将所有应用数据导出为 JSON 文件进行备份，或从备份文件恢复。
*   **UI 设计**：
    *   采用 Jetpack Compose 构建的现代化 UI。
    *   温馨的马卡龙配色风格。
    *   简洁直观的首页数据展示（连续签到、累计签到）。

## 权限说明

为了实现核心的安全监测功能，应用需要申请以下权限：

*   **发送短信 (SEND_SMS)**：用于在紧急情况下自动发送求助短信。
*   **获取位置 (ACCESS_FINE_LOCATION / COARSE_LOCATION)**：用于在报警短信中附带实时位置，方便救援。
*   **忽略电池优化**：确保后台监测服务不被系统杀掉，保证报警功能的可靠性。
*   **开机自启动**：手机重启后自动恢复监测服务。

## 快速开始

1.  **安装应用**：下载并安装 APK。
2.  **初始设置**：
    *   首次启动后，进入“设置”页面。
    *   完善“基本信息”（姓名、联系地址、欢迎语）。
    *   添加“紧急联系人”（至少添加一位）。
    *   授予必要的权限（短信、定位、后台运行）。
3.  **开启监测**：
    *   默认开启手动签到模式。
    *   可在设置中开启“自动签到模式”以获得更无感的体验。
4.  **模拟测试**：
    *   强烈建议在设置页使用“模拟3天未签到报警”功能，测试短信发送和定位获取是否正常。

## 技术栈

*   **语言**：Kotlin
*   **UI 框架**：Jetpack Compose (Material3)
*   **架构**：MVVM (Model-View-ViewModel)
*   **数据存储**：Room Database, Jetpack DataStore
*   **后台任务**：WorkManager, Foreground Service
*   **异步处理**：Coroutines, Flow
